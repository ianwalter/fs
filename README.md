# @ianwalter/fs
> A Proxy over Node's fs module to make it Promise-based and add events

## Features

1. Use Promises and/or async/await with fs methods, e.g.:
   ```js
   const text = await fs.readFile('./top-secret.txt')
   ```

## Installation

```console
yarn add @ianwalter/fs
```
